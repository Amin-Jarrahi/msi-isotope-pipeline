# Output Schema

This document defines the columns and data types for each output CSV file.

---

## `mz_to_mz_isotope_candidates.csv`

Generated by **Stage 1**. Contains one row per (m/z pair, sample) combination.

| Column | Type | Description |
|---|---|---|
| `mz_1` | `float` | First m/z feature (lower value) |
| `mz_2` | `float` | Second m/z feature (higher value) |
| `mz_difference` | `float` | `mz_2 - mz_1` in Daltons |
| `mass_diff_type` | `str` | Matched relationship type: `M+1`, `M+2`, `NH4`, `Na`, or `K` |
| `sample_id` | `str` | Biological sample identifier |
| `combined_score` | `float` | Weighted sum of 9 similarity metrics |
| `score_percentage` | `float` | `combined_score / max(self_score_1, self_score_2) × 100` |
| `mz_1_self_score` | `float` | Self-similarity score for mz_1 |
| `mz_2_self_score` | `float` | Self-similarity score for mz_2 |
| `mz_1_morans_i` | `float` | Moran's I for mz_1 |
| `mz_2_morans_i` | `float` | Moran's I for mz_2 |
| `intensity_ratio_consistency` | `float` | Metric 1 value [0, 1] |
| `value_correlation` | `float` | Metric 2 value [0, 1] |
| `peak_colocalization` | `float` | Metric 3 value [0, 1] |
| `importance_iou` | `float` | Metric 4 value [0, 1] |
| `importance_correlation` | `float` | Metric 5 value [0, 1] |
| `spatial_hist_corr` | `float` | Metric 7 value [0, 1] |
| `value_iou` | `float` | Metric 6 value [0, 1] |
| `morans_similarity` | `float` | Metric 8 value [0, 1] |
| `radial_profile_corr` | `float` | Metric 9 value [0, 1] |

---

## `mz_matching_summary.csv`

Generated by **Stage 1**. One row per sample.

| Column | Type | Description |
|---|---|---|
| `sample_id` | `str` | Sample identifier |
| `total_pairs` | `int` | Number of candidate pairs found |
| `mean_score` | `float` | Mean score_percentage across all pairs |
| `median_score` | `float` | Median score_percentage |
| `high_score_pairs` | `int` | Number of pairs with score > MIN_SCORE |

---

## `identified_isotopes.csv`

Generated by **Stage 2**. One row per confirmed isotope/adduct relationship.

| Column | Type | Description |
|---|---|---|
| `mz_1` | `float` | First m/z (rounded to `HIERARCHY_PRECISION`) |
| `mz_2` | `float` | Second m/z (rounded to `HIERARCHY_PRECISION`) |
| `mass_diff_type` | `str` | Relationship type |
| `n_animals` | `int` | Number of samples where score > MIN_SCORE |
| `mean_score` | `float` | Mean score_percentage across qualifying samples |
| `median_score` | `float` | Median score_percentage |
| `min_score` | `float` | Minimum score_percentage |
| `max_score` | `float` | Maximum score_percentage |
| `std_score` | `float` | Standard deviation of score_percentage |

---

## `identified_isotopes_detailed.csv`

Generated by **Stage 2**. Per-sample detail for confirmed pairs only.

| Column | Type | Description |
|---|---|---|
| `mz_1` | `float` | First m/z |
| `mz_2` | `float` | Second m/z |
| `mass_diff_type` | `str` | Relationship type |
| `sample_id` | `str` | Sample identifier |
| `score_percentage` | `float` | Score for this pair in this sample |
| *(+ all 9 metric columns)* | `float` | Individual metric values |

---

## `parent_children_hierarchy.csv`

Generated by **Stage 3**. One row per parent m/z that has at least one child.

| Column | Type | Description |
|---|---|---|
| `Parent_mz` | `float` | Parent m/z value |
| `Child_1` | `float` or empty | M+1 isotope child m/z |
| `Child_2` | `float` or empty | M+2 isotope child m/z |
| `Child_3` | `float` or empty | NH₄ adduct child m/z |
| `Child_4` | `float` or empty | Na adduct child m/z |
| `Child_5` | `float` or empty | K adduct child m/z |
